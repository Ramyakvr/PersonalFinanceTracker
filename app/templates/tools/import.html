Upload broker statement

Zerodha and Aionion ship tradebook and dividends as two separate XLSX files. Chola ships a single PDF with trades, dividends, and corporate actions interleaved. You can select multiple files per slot (e.g. several periods of Aionion equity-trade exports) — each is imported in turn. Re-uploading the same file is a no-op.

The demat account label is auto-detected from the file's preamble (broker client ID). Multiple files belonging to different accounts of the same broker can be uploaded together — each one is routed to the matching account.
